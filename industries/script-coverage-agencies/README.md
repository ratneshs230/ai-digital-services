# AI-Powered Script Scoring

Automated script analysis that provides instant scores on various elements like plot, character, and dialogue.

## Features

- Interactive multi-step wizard
- AI-powered analysis
- Real-time dashboard
- Chat assistant

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Home page with features overview
- `/tools` - Interactive analysis wizard
- `/dashboard` - Analytics dashboard

---
Built with Next.js 14 + Tailwind CSS
