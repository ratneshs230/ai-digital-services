# AI-Powered Crop Recommendation Engine

Predicts optimal crop inputs (fertilizers, pesticides, seeds) based on soil analysis, weather patterns, and historical yield data to maximize crop yield and minimize waste.

## Features

- Interactive multi-step wizard
- AI-powered analysis
- Real-time dashboard
- Chat assistant

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Home page with features overview
- `/tools` - Interactive analysis wizard
- `/dashboard` - Analytics dashboard

---
Built with Next.js 14 + Tailwind CSS
